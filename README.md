## Задание 1

### Текст задачи

Разработать класс с тремя целочисленными полями. Создать конструктор копирования.
Разработать метод, вычисляющий произведение полей. Перегрузить метод ToString() для
формирования строки из полей класса. Реализовать дочерний класс (его содержание предложить
самостоятельно и описать в решении: какой содержательный смысл имеют поля; реализовать
конструкторы; предложить и реализовать 2-3 метода). Протестировать все конструкторы и
другие методы базового и дочернего классов.

### Алгоритм решения

У нас есть базовый класс Nums с 3 целочисленными полями, у него есть 3 конструктора (пустой, копирующий, с 3 числами), свойства (приватные для set, публичные для get на все элементы), метод перемножения чисел и перегрузка ToString().
У дочернего класса QuadraticEquation добавлены 3 поля float (D - дискриминант, x1-x2 - корни квадратного уравнения). 4 генератора - пустой, с 3 int значениями (a, b, c для рашения уравнения), копирующий для данного класса и для родительского (num1-num3 переходят в a, b, c). В каждом генераторе вызываются приватные методы вычисления дискриминанта и вычисления корней. Больше они не вызываются, поскольку поля - приватные и закрыты на запись, смысла в дополнительном вычислении при этом нет. Есть перегрузка ToString с выводом уравнения в форма ax^2 + bx + c, метод вывода результата (x1 и x2) c проверкой на существование корней уравнения.

-- вычитание минуты из объекта типа Time

int (неявная) – результатом является количество минут (время

bool (явная) – результатом является true, если часы и минуты не равны
