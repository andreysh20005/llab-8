# Шемякин Андрей КМБ-1-2024 Лабораторная №8

## Задание 1

### Текст задачи

Разработать консольное приложение с дружественным интерфейсом с возможностью выбора
заданий для работы с «базой данных (БД)», хранящейся в бинарном файле. Перечень полей
(минимум 5), достаточно полно характеризующих заданную в варианте предметную область,
предложить самостоятельно (постараться отразить в перечне полей такие, которые требуют разных
типов данных). Приложение должно выполнять следующие функции:
1. Чтение базы данных из бинарного файла.
2. Просмотр базы данных.
3. Удаление элементов (по ключу).
4. Добавление элементов.
5. Реализация 4 запросов (формулировки запросов придумать самостоятельно). 2 запроса
должны возвращать перечень, 2 запроса одно значение.
В классе должны присутствовать свойства, конструкторы, перегруженный метод ToString(). Весь
функционал приложения реализовать в виде методов вспомогательного класса с помощью LINQзапросов.
Предусмотреть обработку возможных ошибок при работе программы.

Каталог услуг


### Алгоритм решения
Есть 3 класса - Service, DataBase и Interface.
Service - реализует представление структуры данных, в том числе конструкторы, перегрузку ToString(), а так же метод для ручного заполнения (ввод с клавиатуры всех полей)
Поля:
название услуги, цена услуги, название компании, скидка, доступна ли услуга сейчас
DataBase - реализует все необходимые операции  с БД: чтение и запись из файла, запросы и метод пересчёта цены в цену с учотом скидки.
Interface - реализует взаимодействи с пользователем - методы ввода и вывода строк и чисел с исправлением возможных ошибок, а так же вывод данных из запроса и меню выбора функций для пользователя.

Присутствуют следующие методы для работы с базой данных
ReadAll() - чтение из файла БД в List<Service>
SaveAll() - сохранение из List<Service> в файл БД
GetAllActiveServices() - запрос, результат которого список всех активных услуг
GetWithCurrentDiscount(int discount) - запрос, результат которого список всех услуг с данной скидкой
GetRealPrice() - запрос, результат которого список всех услуг с учётом скидки (поле discount обнуляется)
FindByName() - запрос, возвращающий первую найденную услугу с данным названием
GetServiceRealPrice() - запрос, результат которого - реальная цена для данной услуги (поиск услуги так же по имени)
DeleateByName() - операция поиска и удаления услуг с данным именем
AddService() - операция по добавлению услуги в БД (допускается как передача параметра Service service, так и без параметров, но с вызовом метода для ручного заполнения полей)

Все запросы данных сделаны с использованием query sintax
